Write a program that takes a number as input. If the number is greater than or equal to 100, print 'big number'. If the number is less than 100 and greater than or equal to 50, print 'medium number'. If the number is less than 50, print 'small number'.


Sample Input:
120

Sample Output:
big number


Sample Input:
75

Sample Output:
medium number



Memory limit: 256 MB
Time limit: 15 seconds
